# Hotel Management System – Urban Nest

A console-based Hotel Management System developed in C++ to manage room bookings, customer records, cancellations, and hotel revenue. The system supports different room types, booking history management, room availability tracking, and customer validation using CNIC and username verification.

## Features

- Room Booking System
- Room Availability Management
- Booking Cancellation
- Revenue Tracking
- Booking History Management
- Customer Name & CNIC Validation
- Different Room Categories (Single, Double, Suite)
- Floor & Room Management
- Menu-Driven Console Interface

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Arrays
- Classes & Objects
- Recursion
- File Handling Concepts
- Data Validation

## Project Description

The Hotel Management System simulates a real-world hotel booking environment where customers can reserve rooms based on availability and room type preferences. The system automatically manages room status, tracks bookings, calculates hotel revenue, and handles booking cancellations.

The project demonstrates practical implementation of:
- Object-Oriented Programming
- System Design
- Data Validation
- Booking Management Logic
- Customer Record Handling
